## 배경
• 현대 사회에서 자원의 효율적 활용이 중요해짐에 따라, 렌팃은 소비자들이 물품을 '소유'하는 대신, 필요에 따라 ' 이용'할 수 있도록 돕는 대여 경제 방식을 추구한다.

• 기존의 중고 물품 거래 서비스는 주로 구매와 판매에 초점을 맞추고 있지만, 대여 기능이 부재하여 자원의 순환 활용에 한계가 존재한다.
